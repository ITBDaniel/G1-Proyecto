## 🚀 Cómo iniciar el proyecto

Este proyecto está dividido en componentes de Front-End y Back-End. Sigue estos pasos para ponerlo en marcha localmente:

### 1. Requisitos previos
Asegúrate de tener instalado [Node.js](https://nodejs.org/) en tu equipo.

### 2. Configuración del Back-End
Abre una terminal en la raíz del proyecto y ejecuta los siguientes comandos:

```bash
# Entrar en la carpeta del servidor
cd backEnd

# Instalar las dependencias necesarias (solo la primera vez)
npm install

# Iniciar el servidor en modo desarrollo (con nodemon/live reload)
npm run dev
