## 🚀 Cómo iniciar el proyecto

Sigue estos pasos para instalar las dependencias y levantar el servidor de desarrollo del Back-End:

```bash
# 1. Entrar en la carpeta del servidor backend
cd backEnd

# 2. Instalar las dependencias (solo la primera vez)
npm install

# 3. Iniciar el servidor en modo desarrollo
npm run dev
